### Functions
The function `findJudge` takes one parameter which is the adjacency matrix which is nothing else than the graph
representation of the trust that each citizen has. The function returns the index of the judge otherwise if no judge is
found it returns -1.
### Complexity
The time complexity for the function is `O(n^2)`
